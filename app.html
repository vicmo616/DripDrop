DRIP DROP: A Coding Project

When it counts, you DRIP DROP. A Javascript coding exercise using loops. Write a program that prints the numbers from 1 to 100 and for multiples of '3' display "Drip" instead of the number and for multiples of '5' display "Drop". If a multiple of both display "DripDrop".

Instructions

Enter a Drip and a Drop value. Press the button and the values will appear.
