The Code for DRIP DROP

                        
                            //get values from the user, the drip and drop values
                            function getValues() {
                                let dripValue = document.getElementById("dripValue").value;
                                let dropValue = document.getElementById("dropValue").value;
                            
                                //parse for numbers
                                dripValue = parseInt(dripValue);
                                dropValue = parseInt(dropValue);
                            
                                //check for integers
                                if (Number.isInteger(dripValue) && Number.isInteger(dropValue)) {
                            
                                    //we call dripDrop
                                    let ddArray = dripDrop(dripValue, dropValue);
                            
                                    //we call displayData and write values to the screen
                                    displayData(ddArray);
                            
                                } else {
                                    alert("You Must Enter Integers");
                                }
                            
                            
                            }
                            
                            //do dripDrop
                            function dripDrop(dripValue, dropValue) {
                                let returnArray = [];
                            
                                //loop from 1 to 100
                                for (let i = 1; i <= 100; i++) {
                            
                                    //check to see if divisible by both (3 and 5)
                                    if (i % dripValue == 0 && i % dropValue == 0) {
                                        returnArray.push("DripDrop");
                            
                                    }
                                    //check to see if divisible by drip value(3)
                                    else if (i % dripValue == 0) {
                                        returnArray.push("Drip");
                            
                                    }
                                    //check to see if divisible by drop value (5)
                                    else if (i % dropValue == 0) {
                                        returnArray.push("Drop");
                                    }
                            
                                    else {
                                        returnArray.push(i)
                                    }
                            
                            
                                }
                            
                               return returnArray;
                            }
                            
                            //loop over the array and create a tablerow for each
                            
                            function displayData(ddArray) {
                            
                                //get the table body element from page
                                let tableBody = document.getElementById("results");
                            
                                //get the template
                                let templateRow = document.getElementById("ddTemplate");
                            
                                //clear the table first
                                tableBody.innerHTML = "";
                            
                                for (let index = 0; index < ddArray.length; index += 5) {
                                    let tableRow = document.importNode(templateRow.content, true);
                                    
                                    //grab the td's put into array
                            
                                    let rowCols = tableRow.querySelectorAll("td");
                            
                                    rowCols[0].classList.add(ddArray[index]);
                                    rowCols[0].textContent = ddArray[index];
                            
                                    rowCols[1].classList.add(ddArray[index+1]);
                                    rowCols[1].textContent = ddArray[index+1];
                            
                                    rowCols[2].classList.add(ddArray[index+2]);
                                    rowCols[2].textContent = ddArray[index+2];
                            
                                    rowCols[3].classList.add(ddArray[index+3]);
                                    rowCols[3].textContent = ddArray[index+3];
                            
                                    rowCols[4].classList.add(ddArray[index+4]);
                                    rowCols[4].textContent = ddArray[index+4];
                            
                                    tableBody.appendChild(tableRow);
                                } 
                            
                                //add all the rows to the table
                            }

The Code is structured in three functions.

getValues

getValues is used to get the values that are inputted from the page by the user by use of getElementById. Once that is accomplished we will check to make sure the inputted content is in fact integers by use of parsing the users values with the parseInt function, followed by using an if else statement to verify this as well using the Number.isInteger method which will alert the user that an integer is required if an integer was not used. Once both of the users values are true integers we will call our logic function dripDrop and get its's data back, as well as call our displayData function. Lastly we are to assign our dripDrop function to a variable ddArray where dripDrop accepts two parameters being the dripValue and dropValue.

dripDrop

This logic function is used to return an array returnArray which is assigned a blank array. Next we need to loop from 1 to 100 via a for loop, within the for loop we assign the variable i a value of 1 and incrementing up as long as it is less than or equal to 100. Next we need to check on each number that is being looped through to see if they are divisible by the inputted values from the user. In order to do this we initiate an if else statemnet, the first to check is for both values being divisible by the upcoming number in the loop to print "DripDrop" to maintain the logic of this function before checking the individual values. In the first statement our variable i is checked if its divisible by the value with the modulus operator and seeing if its equal to 0 as well as the users second value, grouping the two statements together using the logical && operator, if the numbers are divisible by both values we then proceed to return our array while pushing a values of "DripDrop" to appear in place of the expected number in the loop using the push method returnArray.push, which appends values to an array. Now we are checking for only one of the users values being the first one within the else if statement, if it is divisible with the upcoming number and pushing "Drip" to appear this time using the push method. Using another else if, we are checking the users second value's divisibility status returning "Drop" instead into the push value with the array. Lastly with our final else statement, if neither value is divisible with the shown number in the loop the value of the number in the variable i is then displayed as a result instead.

displayData

The displayData function accepts the variable ddArray's data as a parameter to display it to the page. ddArray consists of the function dripDrop's values which we assigned to it earlier. We begin with the introduction of the template tag, we input this into the code of our app and what this does is it accepts valid HTML but does not display it. So we embody it with 5 "td" tags and assign it an id of ddTemplate for later reference in our JS code. Next we will grab our table body element from the page. To do this we use again getElementById to get the id "results" and assign it to the variable tableBody. Then we getElementById to get ddTemplate and assign it to templateRow so we can be able to display our template into our table body. The next step is to make sure our table body is clear so we don't attach an excess of data into it that we don't necessarily need. In order to this we take tableBody.innerHTML and set it's value to an empty string. Now since we have an array that is coming in, we need to loop over it and display it, we will be operating on the index, so we will run a for loop and use the length of our ddArray to loop through. We will now declare a new variable called tableRow and assign it with the method document.importNode. What this will do is take the children of our template tag and fragment it, take it and copy it and direct it towards our table body. To make this happen, the templateRow variable, which is equalled to what ddTemplate represents is inputted into the .importNode method and grabs only its content by use of the .content suffix being added to it along with "true" as another parameter so it can grab all of what's associated to it. Next we will now establish how many columns will be in it, determining ultimately how wide we want it to appear. So we will use another selector to count the "td" we have, essentially grabbing them and inserting them into an array and checking its length. The rowCols variable is made to equal our fragment tableRow and use the document method querySelectorAll inserting "td". This instructs our function to take in any and all values that amount to the inserted tag in this instance, and indexing them into an array. The way we do this is is by taking our rowCols current index and making its textContent equal to our ddArray's index that we reference and increase it by use of incrementing it. Before we do this, instead of incrementing by just one in this for loop, we change it to go by 5 to make use of the width we are trying to achieve for our table body. Our final statement will be on each time that we append data into our tableRow we will want to add it to the page, using the document method appendChild inserting then tableRow into its parameter to display its data.
